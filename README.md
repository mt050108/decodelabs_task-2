# DecodeLabs Data Analytics - Project 2: Exploratory Data Analysis (EDA)

## Project Overview
This project focuses on performing Exploratory Data Analysis (EDA) on a dataset using Microsoft Excel. The objective was to understand the data, identify patterns, detect outliers, and generate meaningful insights through summary statistics and visualizations.

## Tools Used
- Microsoft Excel

## Tasks Performed
- Explored the dataset
- Calculated summary statistics
- Created Pivot Tables
- Identified trends and patterns
- Detected outliers
- Analyzed key performance metrics
- Created charts and visualizations (if applicable)

## Key Insights
- The most ordered product is **Printer**.
- The product generating the highest revenue is **Chair**.
- The most commonly used payment method is **Online**.
- Most orders have the status **Cancelled**.
- The most common referral source is **Instagram**.
- The average order quantity is **2.95**.
- No significant outliers were observed in the dataset.
